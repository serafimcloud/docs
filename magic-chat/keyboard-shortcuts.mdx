## Project Shortcuts

| Shortcut | Action |
|----------|--------|
| `Tab` | Switch to next variant |
| `Shift + Tab` | Switch to previous variant |
| `⌘X` / `Ctrl+X` | Copy prompt (when not focused on input) |
| `⌘E` / `Ctrl+E` | Toggle chat panel |
| `⌘B` / `Ctrl+B` | Toggle sidebar |

## Chat Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `⌘Enter` / `Ctrl+Enter` | Submit edited message |

## Variant Management

| Shortcut | Action |
|----------|--------|
| `F2` | Rename current variant |
| `Delete` | Delete current variant |

## General

| Shortcut | Action |
|----------|--------|
| `Escape` | Close dialogs and panels |
