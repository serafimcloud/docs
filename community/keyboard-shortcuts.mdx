## Global

| Shortcut | Action |
|----------|--------|
| `⌘/Ctrl` + `K` | Open command menu |
| `[` | Toggle sidebar |
| `⌘/Ctrl` + `,` | Open settings |
| `Escape` | Close dialog / Exit fullscreen |

## Command Menu

| Shortcut | Action |
|----------|--------|
| `↓` `↑` | Navigate items |
| `Enter` | Select item |
| `Tab` | Switch tabs |
| `Escape` | Close menu |

## Component Page

| Shortcut | Action |
|----------|--------|
| `⌘/Ctrl` + `C` | Copy component code |
| `⌘/Ctrl` + `X` | Copy AI prompt |
| `⌘/Ctrl` + `Shift` + `C` | Copy share link |
| `F` | Toggle fullscreen |
| `C` | Toggle code view |
| `B` | Bookmark |
| `R` | Reload preview |
| `E` | Edit (owner only) |
| `Escape` | Exit fullscreen |

## Preview Dialog (Components)

| Shortcut | Action |
|----------|--------|
| `←` `→` | Previous / Next component |
| `B` | Bookmark |
| `R` | Refresh preview |
| `F` | Open in new tab |
| `⌘/Ctrl` + `X` | Copy AI prompt |
| `Enter` | Open component page |
| `Escape` | Close dialog |

## Preview Dialog (Screens)

| Shortcut | Action |
|----------|--------|
| `←` `→` | Previous / Next screen |
| `Tab` | Switch tabs |
| `B` | Bookmark |
| `Enter` | Add to Canvas |
| `Escape` | Close dialog |

## Tips

<Tip>
**Copy shortcuts** — `⌘C` copies component code only when no text is selected. If you've selected text, it copies your selection instead.
</Tip>

<Tip>
**Shortcuts disabled in inputs** — Keyboard shortcuts don't trigger when you're typing in a search field or text input.
</Tip>
