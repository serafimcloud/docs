# 21st Documentation

Official documentation for [21st](https://21st.dev) — the first design tool for vibe crafting.

## Documentation Structure

- **Getting Started** — Introduction and comparisons
- **Canvas** — AI-powered UI building tool
- **Publishing** — Share components and earn
- **Community** — Browse and use community components

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mintlify) to preview documentation locally:

```bash
npm i -g mintlify
```

Run the following command at the root of your documentation:

```bash
mintlify dev
```

View your local preview at `http://localhost:3000`.
