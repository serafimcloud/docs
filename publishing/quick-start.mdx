## Start

- In Canvas: click **Publish** in the right sidebar
- Or go to **[21st/studio](https://21st.dev/studio)** → **Create new**

## Create Files

- `code.tsx` — main component
- `demos/default/code.demo.tsx` — required demo
- Add dependencies: community components or npm packages

## Add Details

- Name, short description
- Tags
- Upload **Cover image** (required) and **Video** (optional) in Studio

## Publish

<Steps>
  <Step title="Submit for review">
    Click **Publish** to submit for review
  </Step>
  <Step title="Status flow">
    Your component goes through these stages:
    - `on_review` — awaiting review
    - `posted` — approved, available via direct link (not listed)
    - `featured` — promoted and visible in public listings
  </Step>
  <Step title="If not approved">
    Improve quality and **resubmit** for review
  </Step>
</Steps>

## Manage Visibility

Featured components can be set **private/hidden** later in Studio.

## Edit Details after Publish

- In Studio: `⋯` → **Edit details**
- On a component page: top-right `⋯` → **Edit**, or press **`E`** to open the edit dialog
