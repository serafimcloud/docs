## Version History

Every change is saved with previewable versions.

### How to use

<Steps>
  <Step title="Select a prototype">
    Click on the prototype you want to review
  </Step>
  <Step title="Open Versions dialog">
    Open the **Versions dialog** in the sidebar
  </Step>
  <Step title="Browse versions">
    Browse previous versions with previews
  </Step>
  <Step title="Restore">
    Click **Restore** on the version you need
  </Step>
</Steps>

### From chat

Use the **Restore checkpoint** button to roll back to the last saved checkpoint for this prototype.
