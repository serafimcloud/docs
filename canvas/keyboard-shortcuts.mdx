description: Complete reference for Canvas keyboard shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `⌘/Ctrl` + `K` | Command menu |
| `⌘/Ctrl` + `\` | Toggle sidebar |
| `⌘/Ctrl` + `J` | Toggle chat |

## Chat

| Shortcut | Action |
|----------|--------|
| `@` | Mention |
| `Enter` | Send message |
| `⌘/Ctrl` + `Enter` | Start plan build |
| `Shift` + `Tab` | Switch mode |

## Navigation

| Shortcut | Action |
|----------|--------|
| `D` | Design mode |
| `?` | Help & shortcuts |

## Canvas

| Shortcut | Action |
|----------|--------|
| `N` | New variant |
| `U` | Upload image |

## Shape Actions

| Shortcut | Action |
|----------|--------|
| `F` | Focus shape |
| `O` | Preview |
| `I` | Toggle interactive |
| `H` | Version history |
| `⌘/Ctrl` + `D` | Duplicate shape |
| `]` | Move layer up |
| `[` | Move layer down |
| `Backspace` | Delete shape |
