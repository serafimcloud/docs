Canvas is 21st's AI-powered prototyping tool. Build frontend prototypes with AI, explore multiple variations side by side, and get production-ready code you can copy into your project.

## What are Prototypes?

Each frame on your canvas is a **prototype** — a self-contained project where you can build and iterate on ideas. Create multiple prototypes to explore different approaches, compare them visually, and copy the code you like best.

### Sandbox Prototypes

Start with blank prototypes to explore ideas and build new features. Perfect for:
- Trying out UI concepts
- Building components before integrating them
- Experimenting with different approaches

### Connected Repository (Optional)

For advanced use cases, connect your GitHub repository to prototype directly with your existing code and design system. Share live previews with your team to get feedback — without committing anything to your codebase.

<Warning>
  21st doesn't commit directly to your code. Prototypes stay separate until you manually copy the code into your project.
</Warning>
